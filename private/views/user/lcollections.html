Lending Collection

The lending collection has about 5000 books most of them fall into computer science. Using an online library catalog can search for any book.

Reference Collection

The reference collection contains permanent reference materials such as dictionaries, manuals CD collection, encyclopedias. Any material required will be allowed to be photocopied.

UCSC Collection

This collection consists of all the UCSC publications and documents related to the UCSC.
